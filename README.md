# Jumbled Word Game

## 📌 Overview
This is a simple Python-based word game where players guess the correct word from a jumbled version. It helps improve vocabulary and thinking skills.

## 🎮 How it Works
- A word is selected randomly
- The word is shuffled (jumbled)
- User has to guess the correct word
- Score is updated based on correct answers

## 🛠 Technologies Used
- Python

## 🚀 Features
- Random word generation
- Word jumbling logic
- User input handling
- Score tracking

## ▶️ How to Run
1. Install Python
2. Run the file:
   ```bash
   python Jumbled_word_game.py
